## Dependencies

- [Node.js](https://nodejs.org/en/) >= 13.2.0
- [Yarn](https://yarnpkg.com/pt-BR/docs/install)

## Getting started

1. Clone this repository.
2. Run `cd chemistry-mastery`.
3. Run `yarn` to install dependencies.
4. Run `yarn start` to deploy the server.
